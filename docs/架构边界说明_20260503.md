# Architecture Boundary

Atlas Market Engine separates agent-facing choices from rule execution.

## Public Demo Boundary

- Behavior layer: chooses a visible property and an offer in the small demo market.
- Rule layer: checks affordability, records settlement, and updates market state.
- API layer: exposes start, step, status, preset, and report endpoints.
- Web layer: displays the current public-demo state and recent events.

The public edition intentionally uses a compact deterministic flow. It demonstrates the product direction without publishing private production assets.
