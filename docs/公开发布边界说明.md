# Public Scope

Atlas Market Engine is published here as a source-visible noncommercial demo. The repository is meant to show the product direction, architecture boundary, and a reproducible local workflow without exposing private product assets.

## Included

- A lightweight deterministic simulation core.
- CLI, API, and browser dashboard entry points.
- Noncommercial license notice and upstream attribution.
- Minimal tests and a smoke-test path.
- Public documentation for new users.

## Not Included

- Private experiment material or presentation-production assets.
- Commercial parameter packs, tuning records, or production adapters.
- Client, institution, or real-business data.
- Reusable high-value decision templates.
- Local outputs, caches, logs, databases, credentials, or personal files.

The public edition is suitable for evaluation, teaching, and noncommercial experimentation. Commercial use requires separate written permission.
